# ☕ Java Learning Repository

## 📌 Overview

This repository contains all the **core concepts of Java**, organized from beginner to advanced level.
It is structured in a way to make learning and revision easy and systematic.

---

## 📂 Folder Structure

```bash
Java/
├── basics/
├── intermediate-concepts/
├── oops/
└── advanced-concepts/
```

---

## 📖 Contents

### 🔹 basics

Covers fundamental concepts of Java:

* Variables & Data Types
* Operators
* Conditional Statements (if-else, switch)
* Loops (for, while, do-while)
* Input/Output

---

### 🔹 intermediate-concepts

Includes slightly advanced topics:

* Arrays
* Strings
* Methods & Functions
* Collections (List, Set, Map basics)

---

### 🔹 oops

Focuses on Object-Oriented Programming:

* Classes & Objects
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Interfaces

---

### 🔹 advanced-concepts

Covers advanced Java topics:

* Multithreading
* Exception Handling
* File Handling
* Java 8 Features (Streams, Lambda)
* Advanced Collections

---

## 🎯 Purpose

* Build strong Java fundamentals
* Prepare for coding interviews
* Structured learning from basics to advanced

---

## 🚀 How to Use

1. Start from `basics`
2. Move step-by-step to higher folders
3. Practice code alongside concepts
4. Revise using this structured approach
